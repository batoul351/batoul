# 🍽️ Restaurant App

## ✨ Features
- Account creation and email verification only  
- Login functionality  
- Display of categories and meals per category  
- User profile  
- Contact the restaurant  
- Add/remove meals to/from favorites  
- Rate a meal  
- Table reservation  
- Notifications (only via Flutter and Firebase)

## 🛠️ Tech Stack
- **Frontend:** Flutter  
  - State Management: GetX  
  - Networking: HTTP  
  - Local Storage: SharedPreferences  
- **Backend:** Laravel (connected via APIs)
